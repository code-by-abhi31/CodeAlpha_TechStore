# Full-Stack E-Commerce Engine 🛒
Built for the **@CodeAlpha** Software Engineering Internship (Task 1).

### 🚀 Features
* **Live Inventory Simulation:** Background processes deduct stock dynamically to simulate real-world traffic.
* **Smart Specifications:** Dynamically generates technical hardware specs based on item categories using the DummyJSON API.
* **Secure Backend:** Implemented JWT authentication, password hashing, and regex validation for secure user login/registration.
* **Persistent State Management:** Real-time shopping cart and wishlist tracking using MongoDB and local storage.
* **Premium UI/UX:** Custom dark/light mode, toast notifications, and responsive split-screen modals.

### 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+), FontAwesome
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
